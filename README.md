# csvsplitter
A pyhton program to split a CSV File line by line and save each line into a separate CSV file.

Example

Orginal csv file contains

1,2,3,4
5,6,7,8
9,8,9,10
11,12,13,14


It will make a 4 new csv file which will contain each row like
csv1 will contain
1,2,3,4

csv2 will contain
5,6,7,8
